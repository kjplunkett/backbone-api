# First API Project
## Using Backbone w/ an API
### Developing

1. Install Node if you haven't yet: `brew install node` (brew is <a href="http://brew.sh/">homebrew</a>)

2. Install Grunt globally if you haven't yet: `npm install -g grunt`

3. `npm install`

4. `grunt`

5. `localhost:8000`
